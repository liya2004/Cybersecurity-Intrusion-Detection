## Overview
Cyber attacks and unauthorized access are major threats to modern computer networks.  
This project focuses on building an **Intrusion Detection System (IDS)** using machine learning techniques to identify suspicious or malicious network activities.

The model is trained on network traffic data and integrated with a **Streamlit web application** to provide easy and interactive predictions.

---

## Problem Statement
Traditional security systems often fail to detect new and unknown cyber attacks.  
There is a need for an intelligent system that can automatically analyze network traffic and detect intrusions with better accuracy.

---

## Dataset
- The dataset contains network traffic features
- Includes both **normal** and **attack** traffic records
- Used for training and testing the machine learning model
- Data preprocessing and analysis were performed before model training
